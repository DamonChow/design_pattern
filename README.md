## 设计模式

### 构建模式

- Builder （建造者模式）
- Abstract Factory （抽象工厂模式）
- Factory Method （工厂方法）
- Prototype （原型模式）
- Singleton （单例模式）

### 结构模式

- Adapter （适配器模式）
- Bridge （桥接模式）
- Composite （组合模式）
- Decorator （装饰模式）
- Facade （外观模式）
- Flyweight （享元模式）
- Proxy （代理模式）

### 行为模式

- Chain of Responsibility（责任链模式）
- Command （命令模式）
- Interpreter （解释器模式）
- Iterator（迭代器模式）
- Mediator（中介者模式）
- Memento（备忘录模式）
- Observer（观察者模式）
- State（状态模式）
- Strategy （[策略模式]((https://github.com/DamonChow/design_pattern/tree/master/strategy))）
- Template Method （模板方法）
- Visitor（访问者模式）
